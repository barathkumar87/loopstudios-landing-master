Attendance System

The data obtained from the timetable generator is then used to populate the data for the attendance system to be used.

With advancements in fingerprint authenticators to be used in web app we take the modern approach in implementing the attendance system .

The attendance system uses both the user's location for authorization (to check if the student is actually present in the premise) and fingerprint to authenticate.

Besides these it has also got a feature to give the lecturer his/her feedback instantly & anonymously
